# Personalización

Lo primero que hice al acabar el tutorial fue multiplicar por 2 el tamaño del suelo para tener más área de juego. Después dupliqué ese suelo y lo elevé un poco, de modo que solo
puedes acceder mediante una rampa.
También añadí un enemigo extra y un par de coleccionables más.

![Captura de pantalla](Captura%20de%20pantalla%202026-02-05%20221436.png)

En cada extremo del mapa añadí unos portales que te teletransportan al otro extremo.

También añadí un salto al personaje para, en el futuro, poner unas plataformas. Y una zona de muerte, por si te caes del mapa.

Para terminar, le añadí texturas al suelo, el jugador, los objetos coleccionables y a los portales.
